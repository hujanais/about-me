Learning by doing. Projects for desktop applications

Key technologies: C#.NET / WPF

Despite the fact that more and more applications are web or mobile based, there are still applications that are better suited for the desktop. For years I have done full-stack desktop development using the Microsoft frameworks like WCF and WPF. I guess there is still a time and place for it.

BetFair Sports Betting Bot
C#/WPF + MaterialDesign UI
 A bespoke bot for online betting

This is a project I did for a small company in the UK for trading in horse and greyhound racing markets. The bot reads statistical information from their proprietary database to present value bets on the Betfair trading exchange. The connectivity to the Betfair trading exchange is a REST API and a pub/sub streaming socket connection for realtime prices. Apart from the API and algorithm development, I also was tasked at developing a modern look and feel that I chose to use the MaterialDesign WPF port for a more weblike view. This project has the added pressure that customers will be wagering real money on this bot so I will feel their wrath if there are glitches!

Crypto Monitoring Bot
C#/WPF + MaterialDesign UI
 A fun 2-day hackathon for a crypto app for Binance

This is just a fun idea that cropped up talking to a friend about crypto-currencies. We have dabbled in it off and on over the years but never in any meaningful way. For a fun quick project, I thought it would be cool to just write a desktop app that can monitor some technical information about various cyptos. I have a Binance-US account so I just wrote it for this. Initially, I wanted to do this with .net core and run it as a client-server but in the end, with a 2-day time budget, I just went full desktop.
