Kids-Empower-App

This is a full-stack application that was written by myself and a friend of mine. As engineers, we love the idea of building products from soup to nuts and challenge ourselves with all aspects of development just like our work environment but in a setting where we were solely left to fend for ourselves. Pull-request issues, deployment issues, technical issues, whatever it is, figure it out!

This application is essentially a Jira-like task whiteboard for families to encourage children to do chores.

This application encompasses all the fun technical stuff.

  • Node-Express server written in Typescript hosted on Heroku
  • MongoDB database hosted on Atlas
  • AngularJS frontend hosted on Vercel
  • Responsive layout
  • User authentication users using JWT
  • Registration, forgot-password email feature
  • Pair programming sessions on Zoom once per week
