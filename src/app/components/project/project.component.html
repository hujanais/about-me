Learning by doing. Projects in Web and Cloud.

Key technologies: MEAN stack, Docker, NGINX, .NET Core

I have always enjoyed the process of taking something from design to deployment. At first it is just fun and enough to learn bits and pieces of technology but in real-life, the end product in the hands of users should be any project goal. Here I want to be able to deploy full stack or static websites that can actually be consumed by the end user anywhere in the world. These simple but yet practical projects let me practice the fundamentals of AngularJS, Javascript, Node Servers written in both Javascript and Typescipt, Docker containers, NGINX and deployment. Obviously I always find comfort with .NET Core and Visual Studio.

Google Covid-19 Mobility
AngularJS
 Visualizer for Google worldwide mobility data during the 2020 Covid-19 pandemic. The app is written using Angular and hosted on Vercel. The json data was downloaded from Google and then parsed directly into the Angular app. There is no database or Node backend used. visit
WHO Covid-19 Worldwide Data
NodeJS + AngularJS
 Visualizer for WHO worldwide Covid-19 data The initial challenge was to download the situation reports from WHO which are in pdf! Thanks to my son, he wrote a NodeJS application to convert the pdf report into a MongoDB database. I then wrote a small NodeJS based REST server that my visualizer can then just consume. visit
Dryer Temperature and Humidity FullStack with Raspberry PI
MEAN + RaspberryPI + Docker + NGINX
 Monitoring my dryer temperature/humidity using a Raspberry Pi A full MEAN stack application that I use a Raspberry Pi to monitor and upload data from my dryer to a MongoDB database hosted on Atlas. A basic NodeJS server is written that is hosted on Heroku and then visualized using an AngularJS application hosted on Vercel. My family uses this to check on the dry cycle and I also use it to check on the current temperature/humidity since the sensor is mounted at the vent exit close to the outside of the house. visit
Using SenseHat module on RaspberryPI
.NET Core + MongoDB + Visual Studio
 A weather station using a Raspberry Pi This was a quicky 1/2 day challenge to put together a WeatherStation type setup using SenseHat and a RaspberryPI. This time, I wanted to try it with .NET Core and Visual Studio 2019. Wow, what a joy to see intellisense, debugging and compile time syntax checking come alive. I love the MEAN stack but my first love of .NET makes me a bit teary.
