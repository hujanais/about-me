Learning by doing. Projects in Web and Machine Learning

Key technologies: MEAN stack, Python, Docker and Machine Learning
Google Covid-19 Mobility
AngularJS
 Visualizer for Google worldwide mobility data during the 2020 Covid-19 pandemic. The app is written using Angular and hosted on Vercel. The json data was downloaded from Google and then parsed directly into the Angular app. There is no database or Node backend used.
WHO Covid-19 Worldwide Data
NodeJS + AngularJS
 Visualizer for WHO worldwide Covid-19 data The initial challenge was to download the situation reports from WHO which are in pdf! Thanks to my son, he wrote a NodeJS application to convert the pdf report into a MongoDB database. I then wrote a small NodeJS based REST server that my visualizer can then just consume.
Dryer Temperature and Humidity
MEAN + RaspberryPI
 Monitoring my dryer temperature/humidity using a Raspberry Pi A full MEAN stack application that I use a Raspberry Pi to monitor and upload data from my dryer to a MongoDB database hosted on Atlas. A basic NodeJS server is written that is hosted on Heroku and then visualized using an AngularJS application hosted on Vercel. My family uses this to check on the dry cycle and I also use it to check on the current temperature/humidity since the sensor is mounted at the vent exit close to the outside of the house.
Fizz-Buzz Algorithm
Machine Learning
 Solving the FizzBuzz algorithm using Machine Learning in Python FizzBuzz is a simple algorithmic challenge that can be solved analytically. However, I wanted to challenge myself if I can solve the problem by approaching it as a classification problem using a neural network. I was not looking for efficiency or trying to use any novel machine learning methods but rather to understand the ML process by actually solving it with basic ML understanding.
Cart-Pole
Reinforcement Learning
 Solving the classic Cart-Pole physics problem from OpenAI Gym This is the classic intro project for reinforcement learning. I went through with learning to solve this since there are so many example solutions out there.
Tic-Tac-Toe
Reinforcement Learning
 Solving the basic game of Tic-Tac-Toe Ok, this is no Alpha-Go but I now wanted to apply my learning from the Cart-Pole problem to solving Tic-Tac-Toe. Basically the algorithm remained the same and adapted to solve TicTacToe. This is not an exhaustive solution but it did the job.
Connect-4
